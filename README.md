# Liver Cancer Prediction

A machine learning project for **hepatocellular carcinoma (liver cancer) risk prediction** from clinical and laboratory features.

---

## Overview

Liver cancer is one of the most common and lethal cancers worldwide, with late-stage diagnosis dramatically worsening outcomes. This project develops a classification model that predicts liver cancer presence from structured patient data, enabling earlier screening prioritization.

---

## Approach

The pipeline includes:
1. **Exploratory Data Analysis** — distribution analysis, correlation heatmaps, missing value handling
2. **Feature Engineering** — clinical feature selection and normalization
3. **Model Training & Evaluation** — classification algorithms compared on accuracy, AUC-ROC, precision, recall
4. **Interpretation** — feature importance analysis to identify key clinical indicators

---

## Stack

- Python, scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebooks
