Five-year survival for early-stage HCC reaches 70% — but below 10% once it's late, and AFP alone misses up to 60% of early cases. This project stratifies HCC risk from routine lab biomarkers, optimizing for the metric that matters: not missing a cancer.

pipeline
From messy labs to a calibrated risk score.
🧪

Multi-marker features

Liver function, synthetic function, tumour markers (AFP, AFP-L3, PIVKA-II), viral serology and demographics — AFP log-transformed.

🔎

Feature selection

Mann-Whitney screening with BH-FDR, recursive feature elimination, and SHAP for post-hoc importance.

🤖

Model bake-off

Logistic regression, random forest, XGBoost and RBF-SVM, tuned via stratified 5-fold CV with class weighting.

📊

AUROC-first

A false negative costs far more than a false positive — so PR-AUC, sensitivity at the operating point, and calibration all get reported.
